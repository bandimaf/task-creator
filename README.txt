Проект собирается при помощи Gulp (CLI version: 3.0.0, Local version: 4.0.2) и пакета npm (9.3.1).

Для запуска сборки необходимо выполнить следующие команды:

1. Перейти в папку проекта
cd task-creator 

2. Установить глобавльно Gulp CLI
npm install --global gulp-cli

3. Установить зависимости
npm install

4. Запустить проект
gulp start

Дополнительно: сборка проекта – gulp build
